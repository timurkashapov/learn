SparkCore:
- Планирование заданий
- Управление памятью
- Обработка ошибок
- Взаимодействие с системами хранения данных
- ...

SparkCore ---> { RDD API } // Resilient Destributed Datasets/Устойчивые распределённые наборы данных.

RDD API - коллекции элементов, распределенные между множеством вычислительных узлов, котороые могут обрабатываться параллельно.

RDD API ---> { method1(), method2(), ... method3() }
