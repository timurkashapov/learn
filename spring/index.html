1. Введение в Spring

Главная цель фреймворка Spring - упрошение разработки java приложений.

---> Слабая связность посредством внедрения зависимостей.
---> Легковесность и ненасильственность благодаря применению простых Java объектов (POJO).
---> Декларативное программирование через аспекты и общепринятые соглашения.
---> Уменьшение типового кода через аспекты и шаблоны.

Основные особенности Spring:

1. DI (Dependecy injection) - внедрение зависимостей.
2. AOP (Aspect object-programming) - аспектно ориентированное программирование.



Системные службы:

- Безопасность
- Аудит
