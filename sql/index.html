Примеры использования SQL команд

    Выбор всех столбцов в одной таблице: 
  •             SELECT *
            FROM table;
    Выбор одного столбца из таблицы: 
  •             SELECT column
            FROM table;
    Создание псевдонимов столбцов: 
  •             SELECT column1 AS alias1,
                   column2 AS alias2,
                   ...
                   columnN AS aliasN
            FROM table;

Transaction (транзакция) - одна или несколько SQL команд, которые выполняются как единое логическое целое. СУБД определяет трензакцию как неделимую сущность и либо исполняет её, либо нет. Исполнение транзакции гарантирует сохранность данных, а при возникновении ошибки в, хотя бы, одной операции - исполнение транзакции отменятся. View (Представление) -
